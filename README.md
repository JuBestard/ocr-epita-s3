# OCR Princess

## Requirements

Use the SDL library [SDL](https://www.libsdl.org/) to compile this OCR.

For Ubuntu operating systems :

(Ubuntu)
```bash
sudo apt install libsdl1.2-dev
```
(Other) :
Compile this OCR, use a shell.

## Usage

First compile this project by :

```sh
cd src && make
```

To clean this OCR :
```sh
cd src
make clean
```
